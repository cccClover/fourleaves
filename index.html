普通の学校とどこが違う？















大きく違う点は履歴書の学歴欄に、 職業技術専門校に入学、修了したという事は書けないというところです。

私の場合２０２２の三月に大学を卒業、四月に北大阪高等職業技術専門校に入学しましたが

仮に修了した場合も最終学歴は「大卒」ということになります。







授業はどのような感じ？














様々な年齢の方が在籍していますが、授業形式は中学校、高校の授業と大きな違いはありません

また、専攻科目によってクラスが完全に分かれています。

私の場合、北大阪校のＩＣＴプログラミング科に在籍しているのでネットワークやプログラミングの授業を受けています。

プログラムの授業の中で、C言語におけるポインタとアドレスに興味を持ち、授業で学習したコードを

ポインタでを使用した場合はどう書けるか、ライブラリにある関数を使わずに書くにはどうすればいいか

を意識しながらコーディングの自主学習しています。

学習の一例として、改善中のCPUが生成した数字を当てるという簡単なゲームのコードを載せさせていただきます。



                        
---------------------------------------------------------------------------------------------
	#pragma warning(disable:4996)

	#include<　stdio.h　>
	#include<　stdlib.h　>
	#include<　time.h　>

	#define LEVEL 3

	#define LIMIT 7

		make4digit(int a[]);

		int check(int b[]);

		void judge(int c[], int d[], int *jackpot_sub, int *correct_sub);

		void result(int correct_sum, int jackpot_sum);

		int main(void) {
	
			int i_i;
			int chk;
			int jackpot;
			int correct;
			int no[LEVEL];
			int buff[LEVEL];
	
			srand((unsigned int)time(NULL));

			printf("ヌメロンを開始します。 %d個の数字を当ててください。\n",LEVEL);
			printf("1234のように、連続して入力してください。同じ数が含まれることはありません。\n\n");

			make4digit(no);

			do{

				do {

					printf("CPUが数値を生成しました。\nその数値を当ててください\n");
					for (i_i = 0; i_i < LEVEL; i_i++)
					{
					(void)scanf("%d", &buff[i_i]);
					}

					chk = check(buff);

					if (chk == 1) {

						printf("エラーが発生しました。数字を入力しなおしてください。\n");}

					} while (chk != 0);

			judge(buff, no, &jackpot, &correct);
			result(jackpot + correct, jackpot);


			} while (jackpot != LEVEL);

			printf("---Thankyou for playing!!---");
			return 0;
			}

		make4digit(int a[]) {

			int val, i, j;
			
			for (i = 0; i < LEVEL; i++) {
			do {

			val = rand() % LIMIT;
			for (j = 0; j < i; j++) {
				if (val == a[j]) {
					break;
				}
			}

			} while (j < i);

			a[i] = val;
				}
					}

			int check(int b[]) {

			int chk_sum, j, i;
	
			chk_sum = sizeof(b);

			if (chk_sum != LEVEL) {
			return 1;
			}


			for (i = 0; i < LEVEL; i++) {
			if (b[i]<0 || b[i]>(LIMIT - 1)) {
			return 1;
			}

			for (j = 0; j < i; j++) {
			if (b[i] == b[j]) {
				return 1;
					}
				}
			}

			return 0;

			}

		void judge(int c[], int d[], int* jackpot_sub, int* correct_sub) {

			int i, j;

			*jackpot_sub = 0;
			*correct_sub = 0;

			for (i = 0; i < LEVEL; i++) {

			for (j = 0; j < LEVEL; j++) {

				if (c[i] == d[j])
					if (i == j) {
						(*jackpot_sub)++;
					}
				else {
					(*correct_sub)++;
					}
				}
			}
		}

		void result(int correct_sum,int jackpot_sum){ 

			if (jackpot_sum == LEVEL) {

			printf("大正解！");
			}

			else if (correct_sum != 0) {
				printf("%d個の数字は正解しています。\n",correct_sum);

				if (jackpot_sum == 0) {
					printf("しかし、位置がすべて正しくありません。\n");
				}
				else {
					printf("%d個は位置も合っています\n", jackpot_sum);
					}
				}

		}
